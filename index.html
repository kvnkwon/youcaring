    • Objective

      Improve the message of how sharing can impact donations.

      Role

      UI/UX Designer

      • - Conducted user interviews, competitive analyses, and usability tests.
      • - Designed to drive engagement and improve the message of sharing on user campaigns.
      • - Defined hypotheses and test parameters to run A/B tests.

    A/B Testing Experiments

    How can we improve the message of how sharing can impact donations?

    • 1. Leap of faith assumptions

      • Experiment #1
      • Organizers/Donors don’t understand/believe how every share can raise $37.
      • Experiment #2
      • Donors/Visitors won’t share the campaign because they don’t think it will make much of a difference.

    • 2. Hypotheses

      • Experiment #1
      • If we remove the $ amount, the message will seem more clear and trustworthy.
      • Experiment #2
      • If we tailor to a Donor/Visitor in a more personalized way, they might feel more inclined to help.

    • 3. Experiment

      • Experiment #1
      • Remove the $ amount and replace with: "Can't donate? Help by sharing."
      • Experiment #2
      • Personalize sharing message by moving from general statements to messages that invoke “you,” “your share” can help (person name).

    • 4. Behavior

      • Experiment #1
      • Organizer/Donor/Visitor sees message and clicks “Share” button.
      • Experiment #2
      • Donor/Visitor is presented with a pop-up modal upon trying to leave the page, sees a personalized message asking to share, and clicks the “Share” button.

    • 5. Target Metric

      • Experiment #1
      • 5-10% increase in sharing from organizers, donors, and visitors.
      • Experiment #2
      • 10% increase in sharing from organizers, donors, and visitors.

    Proposed Solutions

    Hi-Fi screens that incorporate both the A/B testing experiments and additional UI changes to help support the objective.
